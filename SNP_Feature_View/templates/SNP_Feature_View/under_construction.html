Using Files that aren't Provided as Samples...

The "I have my own file" feature is under construction.... We'd like to allow you to view information from VCF files (and 23andMe files) on your computer without uploading them in full--file uploads take a long time, and we don't like the idea of uploading vast quantities of personal genetic information. Our envisioned solution for quickly and effictively viewing features in your VCF/23andMe file requires reading huge files client-side with JavaScript's FileReader--we don't want you to have to run any additional software outside your browser. We've found JavaScript's native FileReader to be unreliable for files in the 100s of megabytes, and so are working are building a more stable module using FileReader. Questions or suggestions? Contact us or, try the about page! Otherwise go ahead and try SNP See.
